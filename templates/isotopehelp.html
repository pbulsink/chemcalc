Instructions:

The table of isotopes contains exact masses and abundances for all of the stable isotopes in the periodic table.

Updated as new information comes out, it contains information directly from the NIST database.

To view only a single element's isotopes, select it either by name or symbol in the dropdown list at the top. To return to the full list, select "All Elements".
